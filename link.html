Link and Import

HTML 문서 밖의 CSS 파일 링크하기

외부 스타일 시트: .css확장자 파일을 HTML에다 연결하는 방법.

외부 스타일 시트의 장점

  • 여러 HTML 파일들의 스타일 속성들을 동시에 관리 할 수 있으며, 코드의 중복을 방지해주므로 효율적이고 간편해진다.
  • 내부 스타일 시트, 인라인 스타일 보다 CSS의 장점을 효과적으로 쓸 수 있다.
<link rel="stylesheet" type="text/css" href="mystyle.css">

이와 같은 코드를 HTML 문서에 삽입하면 된다.

내부 스타일 시트를 사용하고 싶다면?

HTML 코드 속에다 스타일 태그를 삽입하면 된다.

<style type="text/css"> Selector {property: value;} h1 {color:red} </style>
