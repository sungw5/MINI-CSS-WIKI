Flex

플렉스란?

복잡한 계산 없이 개체들의 레이아웃을 배치 할 수 있는 CSS3의 꿀템입니다. 뷰포트가 불명확한 반응형 디자인에도 간편하게 배치, 정렬, 분산 할 수 있게 도와줍니다.

플렉스의 정의

                
                    .flex_container {
                        display: flex;
                    }

그리드와 마찬가지로 container 과 item에 정의하는 속성으로 나누어집니다.

  • flex container 속성: flex-direction, flex-wrap, justify-content, align-items, align-content
  • flex item 속성: flex, flex-grow, flex-shrink, flex-basis, order

flex container

Property Description
flex-direction flex items의 축(main-axis)을 설정 (기본값: row)
flex-wrap flex-items의 줄묶음(줄 바꿈)을 설정 (기본값: nowrap // nowrap:강제로 한줄에 배치 / wrap: 여러행으로 나눠서 배치)
justify-content 주 축(main-axis)의 정렬 방법을 설정
align-content 교차 축(cross-axis)의 정렬 방법을 설정
align-items 교차 축에서 items의 정렬 방법을 설정

flex-direction

flex의 축 (axis)

flex-wrap

justify-content

align-content

align-items

flex items

Property Description
flex-grow flex item의 증가 너비 비율을 설정
flex-shrink flex item의 감소 너비 비율을 설정
flex-basis flex-item의 (공간 배분 전) 기본 너비 설정
align-self 교차 축(cross-axis)에서 item의 정렬 방법을 설정
order flex item의 순서 설정
