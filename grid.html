Grid

그리드란?

CSS Grid를 통해 웹페이지 내부의 레이아웃 시스템을 손쉽게 구현 할 수 있습니다. 플렉스 박스가 1차원 레이아웃을 도와준다면, 그리드는 2차원에 조금 더 가깝다고 볼 수 있습니다. 그리드와 플렉스와 같은 모듈이 없었다면, 웹페이지 레이아웃 생성은 훨씬 더 복잡하고 어려울 것입니다.

꼭 알아야 할 것

Container(부모) 그리고 그속에 Item(자식) 을 통해 그리드를 표현 할 수 있습니다.

그리드 Cheetsheet 정리

Property Description
grid-template-rows 행 크기
grid-template-columns 열 크기
grid-template-ares 영역 참조 템플릿 생성
row-gap(grid-row-gap) 행과 행 사이의 간격
column-gap(grid-column-gap) 열과 열 사이의 간격
grid-auto-rows 전체적인 행의 크기를 조정
grid-auto-columns 전체적인 열의 크기를 조정
grid-auto-flow row/column/row dense 지정에 따라 자동적으로 배치/td>
justify-content 그리드 콘텐츠들을 수평으로 정렬(오뎅꼬치 가로로 꽂는 것처럼)
align-content 그리드 콘텐츠들을 수직으로 정렬(오뎅꼬치 가로로 꽃은 상태에서 위아래로)
justify-items 그리드 내부개체들을 수평으로 정렬
align-items 그리드 내부개체들을 수직으로 정렬
display: grid 그리드 컨테이너 정의
