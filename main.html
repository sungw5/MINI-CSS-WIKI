Introduction

이 페이지는 무엇인가?

간혹가다 까먹게되는 CSS 요소들을 정리 해놓은 위키 페이지입니다. 종종 헷갈리는 플렉스와 그리드 등등을 정리 해놓았습니다.

CSS 의 적용 우선순위

HTML 은 위에서부터 아래 순서대로 읽지만 CSS 에서는 따로 우선순위가 존재합니다.

  1. 뒤에 !important; 붙인 속성 (;를 꼭 붙여야 함)
  2. 인라인 스타일 방식 (태그에 직접 style 을 넣어서 하는 방식)
  3. 내부 스타일 방식 (HTML 문서에서 직접 style을 넣은 방식)
  4. id 로 지정한 속성
  5. class 와 :(가상클래스) 로 지정한 속성
  6. 그냥 태그 이름으로 지정한 속성
  7. 외부 스타일 방식 (상속된 속성)

강도는 1번이 가장 강함.

만약에 외부 스타일 방식에서 !important 로 지정하면 우선순위가 바뀜.
