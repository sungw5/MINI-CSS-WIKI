Minify

CSS 코드 minify란?

CSS의 코드의 크기를 압축하여 더 작게 만들고 서버와의 데이터 교류간 더 빠른 속도를 위하여 코드를 경량화 시키는 방법입니다.

Minify 예제

  • http://adamburgess.github.io/clean-css-online/
  • https://www.minifier.org/

Minify 전 코드

                
                    1 |  h1{
                    2 |      color: blue;
                    3 |      font-size: 14px;
                    4 |  }

Minify 후 코드

                
                    h1{color:blue;font-size:14px;}
